# README
## Tech Stack
- Language: Ruby 3.0.6
- Framework: Rails 7.0.4
- DB: SQLite

## Prerequisites

- Ruby 3.0.6
- Rails 7.0.4

## Getting Started

- Fork Repo
- Install bundle by running ```bundle install``` 
- ```rails db:create```
- ```rails db:migrate```
- ```rails db:seed```
- ```rails s```


## Unit Tests

- You can run ```rspec``` in Rails Console to run simple unit tests for the project
